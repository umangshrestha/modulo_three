# Modulo Three

## How to run the project locally
Please install [UV](https://docs.astral.sh/uv/getting-started/installation/#standalone-installer) package manager.

```sh
uv venv --python 3.12
source .venv/bin/activate
uv sync
```


## Build the project
```sh
uv build
```

# Run unittest
```sh
uv run pytest
```
